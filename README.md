# UI-test
Minimal React + TypeScript + Vite SPA with Tailwind CSS and ESLint.

## Features
- React 19 + TypeScript with Vite dev server and HMR
- Tailwind CSS, example pages, and reusable components
- ESLint with a sensible project structure

## Requirements
- Node.js 18+ and npm

## Install & Run
```bash
npm install
npm run dev
```

## Build & Lint
```bash
npm run build
npm run lint
```
